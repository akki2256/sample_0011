# Website images

Place your website images in these folders:

- **banner/** – Hero banner images for the homepage (“Creating Impact, Together” section).  
  Use `hero-1.jpg`, `hero-2.jpg`, `hero-3.jpg`.  
  Recommended size: about 1920×1080 px or similar landscape.

- **gallery/** – Photos for the Photo Gallery section and gallery page.  
  Name them as you like and reference them in your HTML/CSS (e.g. `gallery/photo-1.jpg`).

- **team/** – Team member photos for the Our Team section and team page.  
  Name them as you like and reference them in your HTML/CSS (e.g. `team/member-1.jpg`).

Use JPG or PNG. Keep file sizes reasonable (e.g. under 500 KB per image) for faster loading.
